Isogram Printer

Description

This tool will render a page containing the isograms of a function in SVG format.
The function must be a 2D manifold in 3D space of the form z=f(x,y).
The isograms will be with respect to fixed y.

Instructions

  1. Write your function z=f(x,y) in JavaScript.
  2. Enter the domain over which you would like it mapped. Note that the isograms will have fixed y.
  3. Select the domain to pixels ratio. It is recommended to use 300/x_range
  4. Click "Render Isograms" to open a new page with the isogram SVGs
  5. Print the page to get them.
